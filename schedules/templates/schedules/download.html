Meal Master Info Meal Master Criteria:
Copying all meal and wine shifts from the Summer 2017 shift set will meet all the below criteria, if for some reason you can not copy Summer 2017 meal shifts then make sure your shifts match the following criteria in order for them to show up correctly on the Meal Masters document. If you would like to make a permanent change to this set of criteria, please contact the Nate Lohn.

Breakfast Master Criteria:
All Shifts in the master must one of the three flags:
- breakfast
- breakfast-hobart
- breakfast-head

Shifts with the “breakfast” flag must follow these guidelines:
- 2 workers working from 6:45-10:15 (Sunday-Saturday)
- 3 workers working from 7:30-9 (Sunday-Saturday)
- 3 workers working from 7:45-9:30 (Sunday-Saturday)
- 3 workers working from 8:15-10:15 (Sunday-Saturday)

Shifts with the “breakfast-hobart” flag must follow these guidelines:
- 2 workers working from 7:00-10:30 (Sunday-Saturday)

Shifts with the “breakfast-head” flag must follow these guidelines:
- 1 worker working from 6:45-10:15 (Sunday-Saturday)


Lunch Master Criteria:
All Shifts in the master must one of the three flags:
- lunch
- lunch-hobart
- lunch-head

Shifts with the “lunch” flag must follow these guidelines:
- 3 workers working from 11:15-2:00 (Sunday, Tuesday, Wednesday and Friday)
- 2 workers working from 11:15-2:00 (Monday and Thursday)
- 3 workers working from 11:45-1:00 (Sunday, Tuesday, Wednesday and Friday)
- 8 workers working from 11:45-1:00 (Sunday, Tuesday, Wednesday and Friday)
- 4 workers working from 11:45-1:00 (Monday and Thursday)
- 5 workers working from 12:00-2:00 (Sunday, Tuesday, Wednesday and Friday)
- 4 workers working from 12:00-2:00 (Monday and Thursday)

Shifts with the “lunch-hobart” flag must follow these guidelines:
- 2 workers working from 12:00-2:30 (Sunday-Saturday)

Shifts with the “lunch-head” flag must follow these guidelines:
- 1 worker working from 11:15-2:00


Dinner Master Criteria:
All Shifts in the master must one of the four flags:
- dinner
- wine
- dinner-hobart
- dinner-head

Shifts with the “dinner” flag must follow these guidelines:
- 11 workers working from 5:15-8:00 (Sunday-Thursday and Saturday)
- 3 workers working from 6:00-8:00 (Sunday-Thursday and Saturday)
- 9 workers working from 5:15-8:00 (Friday)
- 8 workers working from 5:30-7:00 (Friday)

Shifts with the “wine” flag must follow these guidelines:
- 1 worker working from 5:15-7:30 (Sunday-Saturday)

Shifts with the “dinner-hobart” flag must follow these guidelines:
- 2 workers working from 5:30-8:00 (Sunday-Saturday)

Shifts with the “dinner-head” flag must follow these guidelines:
- 1 worker working from 5:15-8:00 (Sunday-Saturday)
